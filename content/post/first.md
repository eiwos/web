*Full Iframe Widget Api* es un protocolo que permite la comunicacion entre la pagina contenedora y el widget.
Esta formado por un nucleo llamado **Iwa** y unos **extras** que se costruyen sobre **Iwa** para permitir una comunicancion normalizada.

Hemos creado una libreria haxe y compilable de forma nativa en javascript para poder usar el protocolo **FIWA**.

[Puedes visitar la documentacion aqui.](https://beartek.pw/eiwos/docs)
